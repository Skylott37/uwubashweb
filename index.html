Bash

UwUBash is an UwU scripting language like Bash.

Hello world in UwUbash:

source uwu

wwite "Hewwo, wowwd!"

Colored text in UwUbash:

source uwu

wwite "OwO" "$Pink"

Input example in UwUbash:

source uwu

inpwut "Enter your name:" name

wwite "Hello $name"
